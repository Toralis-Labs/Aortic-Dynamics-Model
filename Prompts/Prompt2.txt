It is also very close to how branch ostia are described in vascular geometry work: move along the main-vessel centerline with fine orthogonal slices and identify the slice where the branch just connects to the parent surface; when the centerline itself is the object of interest, the ostium centroid can then be projected back to the parent centerline.

The strongest version is not:

“walk along the branch centerline until a section hits the aorta.”

The stronger version is:

“sample coordinated parent and daughter cross-sections, detect the first true topologic connection event at the ostium, then refine that event into a surface contour and anchor point.”

That distinction matters a lot.

What is strong about your idea

The biggest strength is that it shifts the definition of proximal branch origin away from a single graph node and toward a measured local geometric event. That matches the literature well, because the exact branch point is hard to define precisely, and the ostium boundary or ostium centroid is usually the better fiducial than a pure line divergence point.

the branch has a reasonably tubular proximal segment,
the local centerline tangent is stable,
the branch ostium is not extremely broad,

branch starting point on the branch centerline,
and the ostium centroid projected onto the main-vessel centerline.

the ostium is oblique,
the branch centerline starts slightly distal to the true mouth,
the mouth is elliptical or flared,
or the parent trunk is distorted.

So your idea is anatomically reasonable, but branch-only sampling is not enough by itself.

The exact point of vessel branching is specifically described as difficult when the ostium is large, noncircular, or ambiguous. That is almost a direct warning against using a single cross-section event without refinement.

The same vascular deformation source notes that when fiducial markers get close together and vessel segments become short, measurements get noisier, and segments shorter than about the vessel diameter or less than roughly 5–10 pixels wide are unreliable. If a renal or celiac branch leaves at a shallow angle, a daughter-centered orthogonal cross-section can remain “inside the daughter tube” for several millimeters before reflecting the actual parent attachment. So the first relevant connection may be missed or displaced.

If the input mesh has slight stair-step artifacts, jagged caps, or local bumps, “touching another surface” can happen spuriously.
