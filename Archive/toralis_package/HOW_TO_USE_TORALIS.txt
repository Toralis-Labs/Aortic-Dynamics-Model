Toralis EVAR quick summary

1. Open Terminal.
2. Go to the package folder:
   cd /Users/ahashganeshamoorthy/Desktop/toralis_package
3. Install the package locally:
   pip install .
4. Run the command:
   toralis_evar
5. When prompted, paste the full path to the input .vtp 3D model and press Enter.

What happens next
- The tool runs the orientation step first.
- Then it runs the infrarenal neck measurement step.
- If the model is classified as an abdominal aortic aneurysm (AAA), it writes the measurement outputs.
- If the model is not classified as AAA, it prints: not aortic aneurysm

Default output location
- A new folder is created next to the input file:
  <input_file_stem>_toralis_output

Main output files
- oriented_surface_with_centerlines.vtp
- oriented_labeled_centerlines.vtp
- oriented_labeled_centerlines_metadata.json
- infrarenal_neck_colored.vtp
- infrarenal_neck_report.txt
- toralis_result.txt

Important requirement
- The Python environment must already contain the dependencies needed by the original scripts, especially VTK, NumPy, and VMTK.

Optional direct usage
- You can skip the prompt and run:
  toralis_evar /full/path/to/model.vtp
